File Utility System (CLI Version)
---------------------------------

This is a standalone command-line application that provides various file utilities. It was built using fundamental C programming concepts.

Features:
1. File Encryption / Decryption: Encrypt or decrypt files using XOR or Shift algorithms.
2. Text File Search Engine: Search for words or phrases within a text file and get the exact line numbers.
3. File Merge Utility: Combine multiple files into a single output file.
4. File Organizer Tool: Automatically organize a messy folder by sorting files into subfolders (Images, Documents, Videos, etc.) based on their extensions.

How to Compile:
1. Open your terminal in this directory.
2. Run the `build.bat` script by typing: `.\build.bat`
3. If successful, this will generate `FileUtility.exe`.

How to Use:
1. Run the program from the terminal by typing: `.\FileUtility.exe`
2. Follow the on-screen menu to select your desired utility (type a number and press Enter).
3. Provide the requested file paths and inputs as prompted by the program.
4. Use the "Exit" option from the menu to properly close the application.

Note: This application requires no installation and can be run via the Windows Command Prompt or PowerShell.
